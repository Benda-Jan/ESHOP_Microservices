# EShop_Microservices

Contains:
 - PostgreSQL Database: ORM principle
 - Catalog.API: REST Api, CQRS concept (MediatR), CRUD, Redis Cache
 - EventBus: RabbitMQ
 - Identity.API: JWT Authentication
 - Ocelot Gateway: JWT Authentication Middleware
